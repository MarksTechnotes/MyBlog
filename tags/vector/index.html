🧠 How Big Are Vectors?

Vectors are just arrays of numbers:

Size ≈ Dimensions × 4 bytes

  • 384 dims → ~1.5 KB
  • 768 dims → ~3 KB
  • 1536 dims → ~6 KB

But remember, Total storage includes: Original data + vectors + vector indexes
